🧠 Memory Test — Interactive Cognitive Challenge

Aplicación web interactiva desarrollada con React y Vite que desafía la memoria del usuario a través de un sistema dinámico de emparejamiento de cartas. Diseñada con foco en rendimiento, experiencia de usuario y escalabilidad.

🚀 Highlights
Experiencia fluida con renderizado optimizado gracias a Vite
Interfaz intuitiva y responsiva
Lógica de juego desacoplada y reutilizable
Feedback visual inmediato para mejorar la experiencia del usuario
🛠️ Stack tecnológico
React (Hooks + component-based architecture)
Vite (build tool de alto rendimiento)
JavaScript (ES6+)
CSS moderno
🧩 Features
Sistema de emparejamiento de cartas con lógica de validación
Manejo eficiente del estado del juego
Reinicio dinámico de partidas
Escalabilidad para agregar niveles de dificultad
⚙️ Enfoque técnico

El proyecto fue desarrollado aplicando buenas prácticas de desarrollo frontend, priorizando:

Separación de responsabilidades
Reutilización de componentes
Código limpio y mantenible
Performance en tiempo de desarrollo y ejecución
🎯 Objetivo

Construir una aplicación interactiva que combine lógica, diseño y rendimiento, consolidando conocimientos en React y patrones modernos de desarrollo.
